# Proyecto final - StudyJam

Dentro de este repositorio se encuentra el proyecto fianl de Study Jam, el cual
se trata de un diccionario traductor trilingue para aplicaciones moviles .

## Requerimientos

  * Android Studio 2.3
  * Gradle 2.3
  * minSdkVersion 15

## Uso

Para probar este ejemplo clona este repositorio de la siguiente forma:
>
>     $ git clone https://github.com/RoDriLeoOn/StudyJam_Proyecto_Final.git

Dentro de Android Studio:

* File --> New --> Import Project
* Seleccionas la ruta donde hiciste el `clone` del proyecto.
* Build --> Rebuild Project
* Run
